这个是我的个人博客，用于记录生活，偶尔诗兴大发也会胡乱写点现代诗。

网站使用 Hugo + Github Page 框架构建，文章通过 Github Actions 自动部署。花了几天时间从 Hexo 迁移过来，Hexo 需要太多依赖，不易管理，且生成的时间长，所以就弃用了。随后遇到了Hugo，号称“世界上最快的网站构建框架”，于是就下载试用了一下，果然很快。

Hugo 一般只需几秒钟就能生成一个网站（每页少于 1 毫秒），被称为“世界上最快的网站构建框架”，也使 Hugo 大受欢迎，成为最热门的静态网站生成器之一，被广泛采用。——维基百科·Hugo (软件)

文章内容用 MarkDown .进行编辑，使用 Markdown 可以快速地进行简单的排版。

总之对于我这个 HTML+CSS+JS 都不会的小白来讲，Hugo 真的是上手快，折腾得起。
